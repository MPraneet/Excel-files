<h2>MS Excel based Data Analytics projects.</h2>

<h4>1. Hotel Bookings Cancellations:</h4>
Hotel Bookings Cancellations are analysed by using MS Excel dataset. This Dataset is cleaned, transformed and key factors like guest type, room status of usage and guest arrival count for bookings cancellations are considered.
Required key factors are formulated along with new column additions.
Pivot tables and Data Visualization is done using Pivot charts like Piechart, Barchart, Column charts and Doughnut charts with Pivot slicers.
Given is the Final excel file having dashboard inside Workbook as "Hotel_Bookings.xlsx".

<h4>2. DMart Sales Dashboard:</h4>
DMart Sales and Profit Data is analyzed using MS Excel/csv dataset.
The data is cleaned, transformed and key metrics like category, region, shipmode, segment wise with subcategories are included.
In Sales Dashboard Sheet, mostly Pie charts and Bar chart are used along with Pivot Slicers.
In Profit Dashboard Sheet, mostly a simple Bar chart is used along with Pivot Slicer and Slider for month and year wise profit as per subcategory.

<h4>3. Blinkit Grocery Data Analytics:</h4>
Blinkit Grocery Data Analytics is done for finding Total and Average sales with Count of Items as per Fat content ordered and Outlet data.
The Data is cleaned and keypoints as above help to find out Sales as per Item Type, Outlet Size and Outlet Location.
We find such factors using Pivot Slicers with sum,average,count function to properly calculate every value.
The Dashboard has Icons that switch the Sheets within Workbook by links.
All data is represented using Doughnut charts, Bar charts and Area chart linked with Pivot Slicers to find specific data.

<h4>4. Amazon Sales Dashboard:</h4>
Amazon Sales Dashboard is made for simple representing all data without any manipulations.
Here, we find Total Sales,Product wise Units count, City and State wise Sales amount and thus finding difference between target and actual amount.
The Line chart,Bar chart with Doughnut charts are used to show above parameters.
Links to icons are added to every Sheet for better switching through Sheets in Workbook for easy access.
